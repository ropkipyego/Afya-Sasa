# Nurse Manual

## Purpose

Perform triage, record observations, administer medication, document handovers, and support inpatient/emergency care.

## Triage

1. Open **Triage Queue**.
2. Select waiting patient.
3. Record vitals and triage colour.
4. Submit triage.
5. Patient moves to doctor queue.

## Nursing charting

1. Open **Nursing**.
2. Select active admission.
3. Record vital signs.
4. Schedule medication in MAR.
5. Update MAR status when medication is given/withheld/refused.
6. Add nursing observations.
7. Add shift note.

## Emergency

1. Open **Emergency**.
2. Register emergency presentation.
3. Monitor dashboard and critical alerts.
4. Acknowledge alerts when acted on.

## Safety notes

- Confirm selected admission before recording vitals/MAR.
- Document withheld medication reasons.
- Record shift handovers every shift.
