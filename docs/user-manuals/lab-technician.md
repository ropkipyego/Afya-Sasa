# Lab Technician Manual

## Purpose

Manage lab requests, samples, results, verification, and critical values.

## Workflow

1. Open **Laboratory**.
2. Review lab requests.
3. Collect sample for a request.
4. Enter sample/result details.
5. Verify request when result set is complete.
6. Open **Results Inbox** to monitor critical results.

## Safety notes

- Results are hidden from clinicians until verified.
- Critical values should be verified and communicated urgently.
- Recollection is required for inadequate, haemolysed, clotted, insufficient, or contaminated samples.
