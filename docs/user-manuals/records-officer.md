# Records Officer Manual

## Purpose

Register patients, search existing records, check patients into OPD, and book appointments.

## Patient registration

1. Open **Patient Search**.
2. Search by name, patient number, phone, or identifier.
3. If no match exists, open **Register Patient**.
4. Capture required details:
   - first name
   - last name
   - date of birth
   - gender
   - phone
   - at least one identifier
5. Save the patient.

## OPD check-in

1. Open **OPD Check-In**.
2. Search and select the patient.
3. Enter presenting complaint.
4. Create the OPD encounter.
5. Patient appears in the triage queue.

## Appointments

1. Open **Appointments**.
2. Search and select patient.
3. Enter doctor user ID, date, time, type, and reason.
4. Book appointment.

## Safety notes

- Always search before registering a new patient.
- Do not create duplicate patient records.
- Confirm phone numbers and identifiers carefully.
