# Radiology Technician Manual

## Purpose

Manage imaging requests, reports, verification, and image attachment references.

## Workflow

1. Open **Radiology**.
2. Review imaging requests.
3. Create or update request/report details.
4. Use signed upload helper to get an upload URL for image files.
5. Upload the image to the signed URL outside the app if needed.
6. Attach the uploaded image key/path to the radiology request.
7. Write and verify report.

## Safety notes

- Reports should be verified before clinical use.
- Use descriptive filenames and correct MIME types.
- Do not share signed URLs outside authorised staff.
